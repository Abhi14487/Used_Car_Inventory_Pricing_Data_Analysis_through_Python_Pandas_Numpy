import pandas as pd
import matplotlib.pyplot as plt
A bar chart is perfect for summarizing this relationship between selling price and year as it is clear to compare the difference in avg selling price yearwise. Scatter Chart is more suitable for finding relationship between the two numerical values category. Here, we're comparing categorical values (years) with numerical values (average selling prices). Each bar represents the average selling price for a specific year, making it easy to compare prices across different years.
